Verilen benzer bir düzenle basit bir web sayfası oluşturun. CSS'in temel özelliklerini (display, width, height, margin, padding, border, selector) uygulayın. Sadece HTML5 + CSS3 kullanın, hazir framework kullanmak yasak (Bootstrap, Tailwind vb.).

İçerik

Bu alanda örnek icerik metinleri bulunmaktadir. Css özellikleriyle bu metinler arasinda boşluklar (line-height) ve hizalamalar (text-align) düzenleniriz.

Önce CSS ile kutulari / alanlari yan yana getirmeyi, ardindan alt alta yerleştirmeyi öğrendik. Şimdi ise CSS seçicilerini kullanarak grup ve hiyerarşik seçimler yapmayi göreceğiz.

Yan Menü / Bilgi

Bu alan yan menü veya bilgi kismidir. CSS ile genişlik (width), yükseklik (height), kenar boşlukları (margin/padding) ve kenarlık (border) özelliklerini kullanarak düzenleyebiliriz.

Öğrendiklerimizi hiyerarşik ve grup seçicilerle uygulayip farkii bölümlere stil verebiliriz.
